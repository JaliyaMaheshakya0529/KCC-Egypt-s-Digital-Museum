# KCC Egypts Digital Museum
The Digital Museum of Ancient Egypt

An interactive React web application built for the Web Development challenge outlined. The platform bridges ancient history with a modern, cyber-inspired digital user experience.

---

Tech Stack
- Core Framework: React.js
- Routing: React Router DOM
- Styling: CSS3 (Custom animations, absolute positioning, glow effects)

---

Core Features
- Responsive Dual-Layout Timeline: Automatically switches between a horizontal desktop layout and a vertical mobile table view.
- Modular Exhibition Routes: Configured with client-side routing for Origin, Pyramids, Pharaohs, and Legacy exhibits.
- Artifact Interactive System: The gallery features a dynamic state-driven modal engine to showcase detailed profiles of historical artifacts (e.g., Narmer Palette, Bird Deity).
